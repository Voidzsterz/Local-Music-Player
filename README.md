# Local Music Player

This project is filled with a lot of seemingly random documentation of different C++ methods inside its source.
This is because I'm relatively new to C++ and using this project as a learning tool.
This isn't intended to be a proper Music Player for daily usage.

## Dependencies

CMakeLists.txt is provided. The following libraries are required:

- GLAD
- GLFW
- ImGui
- miniaudio (single header)
- stb_image (single header)
