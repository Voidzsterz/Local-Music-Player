# Local Music Player

UNFINISHED: NOT IN A WORKING STATE

This project is filled with a lot of seemingly random documentation of different C++ methods inside its source. This is because I'm relatively new to C++ and using this project as a learning tool.

This isn't a proper Music Player for daily usage as of right now, I hope in the future it can be however.

Incompatible with Mac and potentially Windows (unsure), may not work on certain Linux distros.

I've made an active attempt to make it compatible with Linux and Windows however, I'm currently debugging this project via a Arch-based Linux distro (Garuda)

I intend to properly debug Windows (and maybe even consider Mac compatibility) when this project is larger.

## Dependencies

CMakeLists.txt is provided. The following libraries are required:

- GLAD
- GLFW
- ImGui
- miniaudio (single header)
- stb_image (single header)
- portable-file-dialogs (single header)
